Setup Guide

Everything you need to put this journal online and turn on the AI features. Two parts, fully self-contained — no separate files to manage.

Part 1 gets your journal live on your own domain. Part 2 connects the AI backend so Market News, the Coach chatbot, and AI quotes work on that live site. You can stop after Part 1 — everything else in the journal already works with no setup at all.

Part 1 — Put Your Journal Online
1
Create a free Netlify account
Go to netlify.com and sign up. No credit card needed.
2
Upload this file
Go to app.netlify.com/drop and drag this very HTML file onto the page. Netlify gives you a live link instantly, like random-name.netlify.app — click it, your journal is now on the internet.
3
Buy your domain
Go to namecheap.com, search nikotradingjournal.com, and buy it (usually $10–15/year).
4
Connect domain to site
In Netlify: your site → Domain settings → Add a domain → type your domain. Netlify shows you some DNS records. Go to Namecheap → your domain → Advanced DNS → add the records Netlify showed you. Wait up to a few hours for it to activate.
✓ Done. Visit your domain — trades, calendar, calculator, knowledge base, and backup/restore all work immediately.
Part 2 — Connect the AI Backend

This turns on Market News, the Coach chatbot, and unlimited AI quotes on your live site. The backend code is built into this journal — just download it below.

1
Get an Anthropic API key
Go to console.anthropic.com → API Keys → Create Key. Copy it immediately (starts with sk-ant-...) — you won't be able to see it again later.
2
Create a free Cloudflare account
Go to dash.cloudflare.com/sign-up.
3
Create a Worker
Workers & Pages → Create → Create Worker. Name it anything, e.g. niko-ai-proxy. Click Deploy.
4
Paste in the backend code
Click Edit code, delete the default template, and paste in the code below. Click Deploy.
5
Add your API key securely
Worker → Settings → Variables and Secrets → Add → type: Secret. Name it exactly ANTHROPIC_API_KEY, paste your key as the value. Save and Deploy.
6
Copy your Worker's URL
It looks like https://niko-ai-proxy.yoursubdomain.workers.dev
7
Connect it here
Scroll to the footer of this journal → click ⚙ AI Backend → paste your Worker URL → Test Connection → Save.
✓ Done. Refresh the page — News, the Coach, and AI quotes now work on your live site.
If Something Doesn't Work
"Test Connection" shows an error — double check the Worker URL was copied exactly with no extra spaces, and that you clicked Deploy after pasting the code.
Error mentioning the API key — go back to Cloudflare → Settings → Variables and Secrets → confirm ANTHROPIC_API_KEY is spelled exactly right.
Domain not loading after a few hours — DNS changes can take up to 24-48 hours. Double-check the records in Namecheap match what Netlify asked for.
